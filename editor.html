範本清單 / 匯入匯出

開啟本頁時會自動載入目前線上的範本（免手動匯入）。匯出是備用；下面的「發布」可直接更新網站、免手動放根目錄。

一鍵發布到網站 免放根目錄

設定一次 GitHub token 後，按「發布」就會直接更新網站的 templates.json，Pages 會自動重建，不用再匯出、手動上傳。
⚠️ token 有寫入權，只在你自己的電腦操作、不要用共用電腦；建議用「fine-grained token」且只授權這一個 repo 的 Contents。
文字座標、字級、描邊、陰影都以「相對影片尺寸的比例」儲存，所以與最終影片解析度無關，燒錄結果會和這裡的預覽一致（WYSIWYG）。
